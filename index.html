Google Docs Desktop for OSX

A Super Simple Google Docs Desktop Client for Mac OSX Built in Javascript and MacGap

Why

Google Docs is an amazing tool for working with word processing documents. Unfortunately, you're limited to the in-browser experience unless you use Chrome. Google Docs Desktop for OSX simply wraps the Google Docs site in an app and allows you to quickly and easily access it as a separate entity outside of having to navigate through your browser tabs.
